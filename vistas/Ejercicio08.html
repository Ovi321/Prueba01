Elemento de diseño de HTML

HEADER: Define un encabezado para un documento o una sección

NAV: Define un contenedor para los enlaces de navegación

ASIDE: Define un Contenido Aparte (Como una barra de tareas)

SECTION 01: Define una sección de un documento

SECTION 02: Define otra sección del documento

ARTICLE: Define un contenedor independiente para un artículo

FOOTER: Define el pie de página del documento
